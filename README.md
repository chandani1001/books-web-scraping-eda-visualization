# Books Web Scraping & EDA

A Python project that scrapes book data from [Books to Scrape](https://books.toscrape.com) using **BeautifulSoup** and **Requests**, stores it in CSV format, and performs **Exploratory Data Analysis (EDA)** using **Pandas** and **Matplotlib** to uncover patterns and insights.

---

## Features

- Scrapes book **titles, prices, ratings, and availability**.
- Saves data into a **CSV file**.
- Cleans and analyzes data using **Pandas**.
- Visualizes insights with **Matplotlib** and **Seaborn**.
- Provides a hands-on example of **web scraping and data analysis workflow**.
